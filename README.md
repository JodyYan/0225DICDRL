# 顏宥庭 | Personal Portfolio

A sleek, modern, and minimalistic personal portfolio website, inspired by the Hostinger Adelina template.

## Features
- **Clean Aesthetic**: Emphasizes premium typography (Playfair Display & Inter) and elegant whitespace.
- **Dynamic Time**: A visually appealing card showing the current local date and time, updating in real-time.
- **Responsive Design**: Adapts gracefully to both desktop and mobile screens.
- **Zero Dependencies**: Built pure and lightweight using HTML5, CSS3, and Vanilla JavaScript.

## Getting Started
There is no build process required. Simply open `index.html` in any modern web browser to view the site instantly!
